# Stellorbit Service

Stellorbit Service is the policy execution and orchestration service for Stellorbit.

Stellorbit's Chinese name is 星轨. The service is responsible for unified orchestration of routing, load balancing, retry, traffic shifting, and service lifecycle governance strategies across distributed systems.

## Responsibilities

- Orchestrate service routing and traffic governance policies.
- Coordinate load balancing, retry, failover, and traffic shifting strategies.
- Manage service lifecycle governance rules and runtime policy state.
- Provide stable service-side APIs for governance engines and platform integrations.
- Expose operational health, metrics, logs, and audit signals for production use.

## Repository Role

This repository contains the service-side implementation for Stellorbit service governance and traffic orchestration.
